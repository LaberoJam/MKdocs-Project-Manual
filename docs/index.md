# Лабораторные работы

### 1. Коммутатор, средства управления и начальная настройка

Эта лабораторная работа знакомит с основными методами конфигурирования сетевого оборудования. В ходе работы студенты научатся подключаться к коммутатору через Web-интерфейс и Telnet, а также выполнят базовые настройки: создание учетных записей, установку времени, изменение IP-адреса и управление портами для обеспечения базовой безопасности и функциональности устройства.

### 2. Виртуальные локальные сети (VLAN)

Эта лабораторная работа посвящена изучению технологии виртуальных локальных сетей для сегментации трафика на канальном уровне. В ходе работы студенты научатся создавать VLAN на основе стандарта 802.1Q, настраивать порты как маркированные (tagged) и немаркированные (untagged), а также использовать функцию сегментации трафика для дополнительной изоляции пользователей внутри одной VLAN.

[:material-arrow-right-bold: Перейти к выполнению](lab2/lab2.md){ .md-button .md-button--primary }

### 3. Протокол STP (Spanning Tree Protocol)

Эта лабораторная работа знакомит с механизмом предотвращения петель в коммутаторных сетях. В ходе работы студенты изучат, как протокол динамически блокирует избыточные каналы связи для построения древовидной топологии без циклов, обеспечивая тем самым отказоустойчивость и защиту сети от сбоев, вызванных ошибками конфигурации.
